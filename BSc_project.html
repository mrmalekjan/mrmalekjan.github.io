BSc Final Project: Smart AC Lamp Dimmer with Closed-Loop PID Control in IoT Framework

University: Amirkabir University of Technology (Tehran Polytechnic)

Field: Electrical Engineering - Control Systems

Supervisor: Dr. Hajar Atrianfar

Completion Date: September 2021

Project Overview

My Bachelor's degree project involved designing and implementing a smart AC lamp dimmer with closed-loop PID control operating within an Internet of Things (IoT) framework. The system enables remote wireless control of lamp brightness through internet connectivity or remote control, while maintaining precise light intensity regulation using PID control with Ziegler-Nichols tuning.

Key Technical Components

  • AC Dimmer Circuit: Designed using TRIAC with leading-edge dimming technique
  • Zero Cross Detection: Implemented for precise phase control timing
  • Light Sensing: BH1750 digital light sensor for accurate lux measurement
  • Microcontroller: NodeMCU (ESP8266) for processing and Wi-Fi connectivity
  • Control Algorithm: PID controller with Ziegler-Nichols tuning method
  • IoT Integration: Thinger.io platform for remote monitoring and control
  • Mobile Application: Android app for user interface and setpoint adjustment

Technical Implementation

Hardware Design

  • Zero Cross Detector circuit for accurate AC waveform detection
  • TRIAC-based dimming circuit with optoisolator for safety
  • BH1750 light sensor with I2C interface for precise light measurement
  • NodeMCU ESP8266 for processing and Wi-Fi connectivity

Control System Design

  • System identification using step response analysis
  • First and second-order system approximations with time delay
  • PID controller design using Ziegler-Nichols methods (both step response and frequency response)
  • Manual PID tuning for optimal performance
  • Implementation of anti-windup protection

Software Development

  • Embedded C++ programming for NodeMCU
  • Interrupt-driven zero cross detection
  • Digital PID controller implementation using MATLAB
  • IoT integration with Thinger.io platform
  • Android mobile application for remote control

Key Achievements

  • Successfully designed and implemented a closed-loop control system for precise light intensity regulation
  • Achieved stable system response with zero steady-state error and settling time under 2 seconds
  • Implemented effective anti-windup protection to prevent integrator saturation
  • Developed complete IoT integration allowing remote monitoring and control
  • Created a user-friendly Android application for seamless interaction with the system
  • Successfully compared different PID tuning methods and selected optimal parameters

Technical Specifications

  • Light Measurement Range: 1-65,535 lux
  • Measurement Accuracy: 0.11 lux
  • Control Frequency: 20 Hz sampling rate
  • Communication: Wi-Fi (HTTP protocol)
  • Dimming Method: Leading-edge phase control
  • Microcontroller: NodeMCU ESP8266
  • Sensor Interface: I2C protocol
