Advanced Machine Learning Course Project

As part of my MSc studies, I undertook a project in the Advanced Machine Learning course where I reproduced and implemented the research paper "An Attention-Based Deep Learning Approach for Sleep Stage Classification with Single-Channel EEG" (IEEE TNSRE, 2021) using Python and PyTorch.

The goal of the project was to simulate the proposed AttnSleep model, an attention-based deep learning framework designed for automatic sleep stage classification from EEG signals.

The implementation required building the full pipeline, including:

  • Multi-Resolution Convolutional Neural Network (MRCNN): to capture both low- and high-frequency features from raw EEG signals.
  • Adaptive Feature Recalibration (AFR): to enhance feature representation by modeling inter-dependencies between extracted features.
  • Temporal Context Encoder (TCE): to model temporal dependencies in sleep patterns using multi-head self-attention with causal convolutions.
  • Class-Aware Loss Function: to address class imbalance commonly present in physiological sleep datasets.

I trained and tested the model on publicly available EEG datasets (such as Sleep-EDF), evaluated its performance against baseline methods, and confirmed that the reproduced model achieved comparable results to the reported benchmarks.

Through this project, I gained extensive hands-on experience with deep learning for time-series biosignals, attention mechanisms, and practical challenges such as imbalanced medical data, model optimization, and reproducibility of research code.

This project strengthened my expertise in applying advanced deep learning architectures to healthcare-related signal processing problems and gave me practical insights into bridging theoretical research with implementation.
