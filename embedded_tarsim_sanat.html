    • Projects History

    1. SiVo

    Elevator speaker system that can play music, announce floors, and connect directly to the elevator phone. Embedded Systems: WCH/ST microcontrollers and Raspberry ICs. Peripherals: UART, SPI, GPIO, TIMER, EXTI Operating System: Free-RTOS

    2. SiAx

    Elevator access control system using RFID technology and announcing floors and people authorized to access, with direct connection to the Sidex system Embedded Systems: WCH/ST microcontrollers and Raspberry ICs. Peripherals: UART, GPIO, TIMER, EXTI, I2C, SPI Operating System: Free-RTOS

    3. Sidex

    Main part of the access control system, consisting of GSM communication, SD card, OLED display, Speaker, Rotary Switch and Push Buttons, Ethernet connections, Different Power up modes (In plug and battery operated) and most important, 5 powerful and various microcontrollers that work in coordination. Embedded Systems: WCH/ST/ESP microcontrollers. Peripherals: USART, UART, GPIO, TIMER, EXTI, I2C, SPI, CAN, ADC, DAC Operating System: Free-RTOS

    4. Destination Control System Simulator

    A graphical interface made using Node.js that connects to verify Sidex performance and check sample data logs through it and also to make random datas for training destination system AI algorithms

    5. SiTel Echo+

    A touch-screen telephone that can connect to the telephone network as well as communicate directly with the SiAx system. Embedded Systems: WCH/ST microcontrollers. Peripherals: UART, GPIO, TIMER, EXTI, I2C, ADC Operating System: Free-RTOS

    6. SiHall

    System connecting the elevator panel to the Sidex system via stable and secure CAN or ModBus communication Embedded Systems: WCH/ST microcontrollers. Peripherals: CAN, UART, GPIO, TIMER, EXTI, I2C, DAC Operating System: Free-RTOS
