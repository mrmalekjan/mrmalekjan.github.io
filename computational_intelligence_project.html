Fuzzy Control of a Three-Link Gymnastic Robot

Computational Intelligence Course Project

As part of my Bachelor's degree course in Computational Intelligence, I designed and implemented a fuzzy logic controller for a three-link underactuated robotic system to perform a gymnastic "giant circle" maneuver.

The project involved a comprehensive process, beginning with the dynamic modeling of the robot using Lagrangian mechanics to derive the governing equations of motion. The resulting nonlinear system was simulated in MATLAB/Simulink to validate the model.

The core of the project was the design of a Mamdani-type fuzzy logic controller (FLC) to generate the control torques required for the swing motion. Two separate fuzzy inference systems (FIS) were developed, each taking the error and the derivative of error for a specific joint angle as inputs. The FISs utilized triangular membership functions, a set of empirically tuned fuzzy rules, and the centroid method for defuzzification.

The simulation results successfully demonstrated that the designed fuzzy controller could effectively drive the robot to perform the continuous 360-degree giant circle motion. The control torques remained within the specified physical limits, and the graphical animation confirmed the desired kinematic behavior.
