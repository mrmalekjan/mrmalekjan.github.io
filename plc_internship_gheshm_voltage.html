PLC and HMI Programming Internship

Duration: 5 months (May 2020 - October 2020)

Role: PLC & HMI Programming Intern

Supervisor: Eng. Nouri

Academic Advisor: Dr. Mohammad Azam Khosravi

Internship Overview

My internship focused on learning and working with various PLC systems including Siemens S7 series (S7-1500, S7-1200, S7-300) and Kinco K5 series, along with their corresponding HMI programming. I utilized industry-standard software tools like TIA Portal (Totally Integrated Automation) and KincoBuilder for programming and simulation.

Key Technologies Used:

  • Siemens S7-1500, S7-1200, S7-300 PLCs
  • Kinco K5 Series PLCs
  • Kinco MT4000 and MT5000 HMI Series
  • Siemens TIA Portal V16
  • SIMATIC STEP 7
  • KincoBuilder
  • Kinco HMIware

Technical Skills Acquired

PLC Programming & Configuration

  • Hardware Configuration: Learned to configure various PLC components including Power Modules (PM), Power Supply (PS), CPU modules, and signal modules
  • Ladder Logic Programming: Mastered programming in LAD (Ladder Diagram) language including:
    • Bit logic operations (contacts, coils, set/reset)
    • Flip-flops (SR, RS)
    • Timers (TON, TOF, TP, ODT, ODTS, OFFDT, PULSE)
    • Counters (CTU, CTD)
    • Comparators
    • Edge detection (F_TRIG, R_TRIG)
  • Advanced Programming: Implemented Function Blocks (FB), Data Blocks (DB), and PID controllers
  • Memory Management: Worked with different memory types (Load Memory, Work Memory, Retentive Memory)

HMI Development

  • Designed user interfaces for industrial applications using Kinco HMIware
  • Implemented various HMI elements:
    • Bit State Setting & Bit State Lamp
    • Bit State Switch
    • Multiple State Setting & Display
    • Number Input & Display
    • Direct & Indirect Windows
    • Trend Curves for data visualization
  • Created multi-page HMI applications with navigation between screens

System Integration & Communication

  • Configured PROFINET and PROFIBUS communication protocols
  • Worked with GSD files for integrating third-party devices
  • Implemented PWM/PTO for motor control applications
  • Configured High-Speed Counters (HSC) for frequency measurement

Responsibilities, Tasks and Projects

Siemens S7-1500 System Implementation

Comprehensive study and implementation of Siemens S7-1500 PLC systems, including:

  • Hardware component analysis (PM, PS, CPU, memory cards, additional modules)
  • Programming in TIA Portal environment
  • Simulation and testing of control logic
  • Force functionality for testing and troubleshooting
  • Protection mechanisms implementation

PID Controller Implementation

Designed and implemented PID control systems using:

  • Cycle interrupt organization blocks for consistent control loop timing
  • PID_Compact function blocks in TIA Portal
  • Configuration of controller parameters and process value settings
  • Simulation and monitoring of control performance

Motor Control Applications

Developed control systems for various motor types:

  • PWM (Pulse Width Modulation) implementation for speed control
  • PTO (Pulse Train Output) configuration for stepper motor control
  • Motion control using technology objects in TIA Portal
  • Commissioning and manual control interfaces

Input Choke Specification

Performed technical calculations and selection of input chokes for inverters:

  • Calculated maximum current requirements (FLA × 150% = 453A)
  • Evaluated appropriate choke models based on technical specifications
  • Considered parameters like rated current, adapted power, and pressure drop

Outcomes & Impact

This internship significantly enhanced my understanding of industrial automation systems and provided me with:

  • Problem-Solving Skills: Developed systematic approaches to troubleshooting automation systems
  • Technical Documentation: Learned to create comprehensive technical reports and documentation
  • Industry Standards: Gained familiarity with industrial communication protocols and standards
  • Project Management: Understood the complete lifecycle of automation projects from design to implementation
  • Safety Awareness: Learned about safety considerations in industrial control systems
